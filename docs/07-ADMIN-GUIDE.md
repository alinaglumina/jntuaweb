# Admin User Guide

For staff who manage content. No technical knowledge required.

## Signing in
1. Go to `https://<your-site>/admin`.
2. Enter your username and password.
3. On first login you'll be required to set a new password.
4. Forgot your password? Use "Forgot password" on the login page; a reset link is
   emailed to you (valid 60 minutes).

## The dashboard
After login you land on the dashboard: quick stats and recent activity. The left
sidebar groups every section under **Content, Institutional, Academics, System,
Communication, Administration,** and **Account**.

## Managing content (any section)
Every content section works the same way:
- **Add** — click *Add*, fill the form, Save.
- **Edit** — click the pencil on a row.
- **Delete** — click the trash icon. Deleted items go to **Trash**, not gone
  forever.
- **Trash / Restore** — toggle **Active / Trash** at the top right. In Trash you
  can **Restore** an item or **Delete forever**.
- **Search** — the search box filters the current section (server-side).
- **Pages** — long lists are paginated (25 per page).

Rich-text fields have a formatting toolbar (bold, lists, links, headings). Image
and file fields let you upload directly; images are automatically compressed.

## Quick search (⌘/Ctrl + K)
Press **Ctrl+K** (or ⌘K on Mac) anywhere in the admin to open the command
palette. Type to **jump to any section** or **find records** across content types.

## Media Library
Under **Media**:
- **Upload** files (drag-and-drop multiple, with a progress bar) — PDF, Word,
  Excel, PowerPoint, images, video, ZIP.
- **Preview** any file (images, PDFs, video play inline; Office docs render via
  Office Online).
- **Replace** a file's contents while keeping the same link everywhere.
- **Download** (tracked — counts show on each file).
- **Bulk actions** — select multiple files to move or delete at once.
- **Download report** — see your most-downloaded files and totals.

## Notifications & news
Add notices under the relevant section; they appear in the public Notification
Centre, filtered by category. Attach a PDF/file where needed.

## Users, roles & security
Under **Administration**:
- **Users** — create accounts; set a temporary password and require change on
  first login.
- **Roles & Permissions** — 14 roles; directors are scoped to their directorate.
- **Login History** — see recent sign-ins and active sessions; **revoke** a
  session or "sign out all other sessions".
- **Audit Logs** — every change is recorded (who, what, when).

## Settings & SEO
- **Settings** — site-wide options.
- **SEO** — set per-page title/description/keywords/OG image; these populate the
  page's meta tags automatically.
- **Page Builder / Menu Manager** — edit content blocks and header/footer menus.

## Backup
Backups run automatically every night on the server. **Backup** in the admin lets
you download an on-demand JSON snapshot. (Full server backups are managed by the
IT team — see the Maintenance Guide.)

## Good habits
- Change your password regularly; never share accounts.
- Prefer **Trash** over "delete forever" so mistakes are recoverable.
- Add descriptive titles — they power the site search.
